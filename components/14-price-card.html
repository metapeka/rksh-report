14 · PriceCard · Страничный

Карточка тарифа: заголовок, цена, описание, список преимуществ.

→ Код: src/components/ui/PriceCard.astro → Showcase: /styleguide

Состав

PriceCard = div.card + h3.title + div.price + p.desc + ul.highlights

Использование (Astro)

---
import PriceCard from '../components/ui/PriceCard.astro';
---

<PriceCard
  title="Начальная школа 1–4 классы"
  price="66 880 р./мес"
  highlights={['3–4 урока ежедневно', 'В классе до 15 человек', 'Продлёнка с питанием']}
/>

<PriceCard
  title="Средняя школа 5–8 классы"
  price="66 880 р./мес"
  description="5–8 уроков ежедневно"
  highlights={['В классе до 15 человек']}
/>

price string — Стоимость (обязательный)
description string — Краткое описание
highlights string[] [] Список преимуществ (с галочкой ✓)

CSS-справка

Это справочник. Единая точка правды — src/components/ui/PriceCard.astro.

.price-card              /* корень: bg-surface, border, radius-xl, padding-2xl */
.price-card:hover        /* box-shadow: shadow-md */
.price-card__header      /* margin-bottom: lg */
.price-card__title       /* heading font, xl, primary */
.price-card__price       /* heading font, 2xl, bold, accent-warm */
.price-card__desc        /* muted, base, relaxed */
.price-card__highlights  /* ul: no list-style, no padding */
.price-card__highlights li /* relative, padding-left: lg, checkmark ✓ */

Ключевые токены:

  • border: 1px solid var(--color-primary-line)
  • border-radius: var(--radius-xl)
  • padding: var(--spacing-2xl) → мобилка var(--spacing-xl)
  • checkmark: color: var(--color-success), font-weight: var(--font-weight-bold)
  • price: --font-size-2xl, --font-family-heading, --color-accent-warm
