Видеоархив (/videoarhiv)

Статус: 🔨 В работе

Видеоархив — вкладка «Видео» на странице /media. Отдельной страницы /videoarhiv нет.

→ Общие правила: 01-design-guide.md → Родительская страница: src/pages/media.astro

Контент

Источник данных

Данные Откуда
Видео CMS media_videos collection
Категории Вычисляются из category поля записей ([...new Set(videos.map(v => v.data.category))])

CMS коллекция media_videos

Seed-схема (устарела — не совпадает с кодом страницы):

Поле (seed) Тип Обязательное Описание
title string ✅ Заголовок
youtube_id string ✅ YouTube ID
description text ❌ Описание

Поля, используемые в коде страницы (фактические):

Поле (код) Тип Описание
title string Заголовок → VideoCard title
url string Ссылка на видео → VideoCard url
thumbnail image Превью → thumbnail.src для VideoCard thumbnailUrl
category string Категория для фильтрации

⚠️ Расхождение: seed-схема содержит youtube_id + description, а страница использует url + thumbnail + category. Схему нужно обновить через Admin UI.

Вкладка видео скрыта по умолчанию (.media-videos--hidden { display: none }). Переключение через JS-обработчик на .media-tabs__btn.

Компоненты

Компонент Файл Где на странице
VideoCard ui/VideoCard.astro Все видео из media_videos
Button ui/Button.astro CTA «Записаться на экскурсию» (variant=warm, size=lg)

VideoCard props

Prop Тип Описание
title string Название видео
url string Ссылка (внешняя, target="_blank")
thumbnailUrl string URL превью (v.data.thumbnail?.src)
thumbnailAlt string Alt-текст превью

VideoCard — ссылка (<a>) с превью 16:9, play-иконкой по hover (абсолютный оверлей) и заголовком внизу.

Уникальные стили

Класс Назначение
.media-videos Секция видео (padding 2xl→4xl)
.media-videos--hidden display: none (скрытая вкладка)
.media-grid--videos Grid 3× видео (gap xl; 1× на мобилке)
.media-grid__item Обёртка VideoCard (data-category для JS-фильтра)
.media-grid__item--hidden display: none (скрыт фильтром)

Остальные стили общие с фотогалереей — см. 11-fotogalereya.md.

JS-логика (встроена в media.astro)

  1. Табы: клик по .media-tabs__btn переключает видимость #tab-photos / #tab-videos через toggle класса .media-videos--hidden
  2. Фильтры: клик по .media-filters__btn фильтрует .media-grid__item по data-category (toggle .media-grid__item--hidden)
