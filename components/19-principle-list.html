19 · PrincipleList · Страничный

Нумерованный список принципов обучения с 3 вариантами отображения.

→ Код: src/components/ui/PrincipleList.astro → Showcase: /styleguide

Состав

PrincipleList = ol > li × N (варианты: accent, timeline, minimal)

Каждый элемент: номер/маркер + заголовок + описание.

Варианты

accent (по умолчанию)

Оранжевый круг с номером + заголовок (оранжевый, bold) + описание. Разделитель — border-bottom.

timeline

Вертикальная линия слева + синие круги с номерами + заголовок + описание.

minimal

Маркер ● (оранжевый) + заголовок (semibold) + описание. Лёгкий разделитель.

Использование (Astro)

---
import PrincipleList from '../components/ui/PrincipleList.astro';

const principles = [
  { title: 'Принцип сознательности', description: 'Механическое заучивание недопустимо' },
  { title: 'Принцип системности', description: 'Знания в строгой последовательности' },
  { title: 'Принцип постепенности', description: 'Переход после усвоения предыдущего' },
];
---

<PrincipleList principles={principles} />
<PrincipleList principles={principles} variant="timeline" />
<PrincipleList principles={principles} variant="minimal" />

CSS-справка

Это справочник. Единая точка правды — src/components/ui/PrincipleList.astro.

.principle-list                  /* ol: no list-style, no padding */
.principle-list--accent         /* гибкий список с border-bottom */
.principle-list--timeline        /* вертикальная линия слева, padding-left */
.principle-list--minimal         /* лёгкий список с ● маркером */
.principle-list__item            /* li: flex/timeline/padding */
.principle-list__num             /* accent: оранжевый круг 2rem, white, bold */
.principle-list__marker          /* timeline: синий круг 2rem, surface text */
.principle-list__body            /* flex column, gap 2xs */
.principle-list__title           /* base, semibold, text */
.principle-list__title--accent  /* accent: warm, bold, base */
.principle-list__desc            /* sm, muted, relaxed */

Ключевые токены:

  • accent num: background: var(--color-accent-warm), color: #fff, 2rem, border-radius: 50%
  • timeline num: background: var(--color-primary), color: var(--color-surface), 2rem
  • timeline line: 2px solid var(--color-primary-line)
  • minimal marker: ●, color: var(--color-accent-warm)
  • Адаптив (≤768px): тоньше маркеры, меньше padding
