11 · TeacherCard · Составной

Карточка преподавателя: аватар (фото или инициалы), ФИО, должность, описание.

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/TeacherCard.astro → Showcase: /styleguide

Состав

TeacherCard = Card (surface) + Avatar + Name + Position + Description

Использование (Astro)

---
import TeacherCard from '../components/ui/TeacherCard.astro';
---

<!-- С фото -->
<TeacherCard
  name="Мишина Антонина"
  position="Учитель-логопед"
  photoUrl="/uploads/mishina.jpg"
  photoAlt="Мишина Антонина"
/>

<!-- Без фото — покажет инициалы -->
<TeacherCard
  name="Мишина Антонина"
  position="Учитель-логопед"
/>

<!-- С описанием (PortableText рендерится отдельно на странице) -->
<TeacherCard
  name="Черткова Ирина"
  position="Учитель начальных классов"
  description="Опытный педагог с многолетним стажем..."
/>

Пропсы

Проп Тип По умолчанию Описание
name string — ФИО преподавателя (обязательный)
position string — Должность
description string — Краткое описание (опционально)
photoUrl string — URL фото. Если пусто — показываются инициалы
photoAlt string '' Alt-текст для фото
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/TeacherCard.astro.

.tcard              /* корень — Card с text-align: center */
.tcard__avatar      /* круг 80×80, bg-primary-soft */
.tcard__avatar img   /* object-fit: cover */
.tcard__initials    /* инициалы — bold, primary, 2xl */
.tcard__name        /* h3 — heading, lg */
.tcard__position    /* должность — muted, sm */
.tcard__desc        /* описание — text, sm, left-aligned, relaxed line-height */

Ключевые токены:

  • avatar: 80×80px, border-radius: 50%, background: var(--color-primary-soft)
  • initials: --font-size-2xl, --font-weight-bold, --color-primary
  • name: --font-size-lg, --font-family-heading
  • position: --font-size-sm, --color-text-muted
  • description: --font-size-sm, --color-text, text-align: left, --line-height-relaxed
