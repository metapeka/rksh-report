29 · OverviewCardsNav — карточки-обзор навигация

Сетка мелких карточек для навигации внутри секции (например, подкурсы).

Расположение

src/components/ui/OverviewCardsNav.astro

Пропсы

Проп Тип Обязательный Описание
items { title: string; meta?: string; href: string }[] ✅ Список пунктов
class string ❌ Дополнительный CSS-класс

Визуал

  • Сетка: grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)), gap --spacing-md
  • Карточка: фон --color-surface, граница --color-primary-line, padding --spacing-md, border-radius --radius-lg
  • Hover: border-color: --color-accent, box-shadow: --shadow-md, translateY(-2px)
  • Title: font-weight-semibold, font-size-sm, color: --color-text
  • Meta: font-size-xs, color: --color-text-muted

Использование

<OverviewCardsNav
  items={[
    { title: 'Подготовка к школе', meta: '4–7 лет', href: '/doshkolniki/podgotovka-k-shkole' },
    { title: 'Шахматы', meta: 'от 5 лет', href: '/doshkolniki/shahmaty' },
  ]}
/>
