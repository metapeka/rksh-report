16 · ServiceTable · Страничный

Таблица услуг с произвольными заголовками и строками. Последняя колонка подсвечивается как цена.

→ Код: src/components/ui/ServiceTable.astro → Showcase: /styleguide

Состав

ServiceTable = div.wrapper + div.scroll > table (произвольные заголовки + строки)

Использование (Astro)

---
import ServiceTable from '../components/ui/ServiceTable.astro';
---

<ServiceTable
  headers={['Наименование услуги', 'Формат', 'Стоимость']}
  rows={[
    ['Постановка звуков (4,5–9 лет)', 'Индивидуально', '30 мин — 1 800 р.'],
    ['Логопедическое обследование', 'Индивидуально', '1 час — 4 400 р.'],
  ]}
/>

Пропсы

Проп Тип По умолчанию Описание
headers string[] — Массив заголовков колонок (обязательный)
rows string[][] — Массив строк, каждая — массив ячеек (обязательный)
class string '' Дополнительные CSS-классы

Примечание: Последняя колонка в каждой строке автоматически подсвечивается оранжевым (--color-accent-warm, font-weight: semibold, white-space: nowrap).

CSS-справка

Это справочник. Единая точка правды — src/components/ui/ServiceTable.astro.

.service-table              /* корень: margin-bottom 2xl */
.service-table__scroll      /* overflow-x: auto */
.service-table__table       /* width: 100%, border-collapse */
.service-table__table th    /* bg-primary, text-surface, semibold, nowrap */
.service-table__table td    /* border, padding sm md, text-align: left */
.service-table__price       /* последняя ячейка: accent-warm, semibold, nowrap */

Ключевые токены:

  • th: background: var(--color-primary), color: var(--color-surface)
  • чётные строки: background: var(--color-surface-alt)
  • цена (последняя колонка): color: var(--color-accent-warm), font-weight: var(--font-weight-semibold)
  • font-size: var(--font-size-sm) для всей таблицы
