27 · Icon — универсальная SVG-иконка

Централизованный компонент иконок. 3 стиля рендеринга, 22 иконки, маппинг для SubjectGrid.

Расположение

  • Компонент: src/components/ui/Icon.astro
  • Словарь иконок: src/icons/index.ts

Пропсы

Проп Тип Обязательный По умолчанию Описание
name IconName ✅ — Ключ из словаря иконок
style `'outline' 'filled' 'duotone'` ❌
size number ❌ 24 Размер в px
class string ❌ — Дополнительный CSS-класс

Стили рендеринга

Стиль SVG атрибуты Визуально
outline stroke="currentColor" stroke-width="1.5" fill="none" Линейная иконка
filled fill="currentColor" Заливная иконка
duotone bg-слой opacity="0.2", fg-слой fill="currentColor" Двухцветная

Доступность

Все иконки имеют aria-hidden="true" — они декоративные. Текст рядом с иконкой должен быть отдельным элементом.
