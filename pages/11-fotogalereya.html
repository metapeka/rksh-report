Фотогалерея (/fotogalereya)

Статус: ✅ Готова (код + схема совпадают, ожидается контент)

Фотогалерея — вкладка «Фотогалерея» на странице /media. Отдельной страницы /fotogalereya нет.

→ Общие правила: 01-design-guide.md → Родительская страница: src/pages/media.astro

Контент

Источник данных

Данные Откуда
Фотографии CMS media_gallery collection
Категории Вычисляются из category поля записей ([...new Set(photos.map(p => p.data.category))])

CMS коллекция media_gallery

Seed-схема (устарела — не совпадает с кодом страницы):

Поле (seed) Тип Обязательное Описание
image image ✅ Изображение
alt string ❌ Alt текст
caption string ❌ Подпись

Поля, используемые в коде страницы (фактические):

Поле (код) Тип Описание
title string Заголовок → GalleryCard title
image image Изображение → image.src для GalleryCard imageUrl
category string Категория для фильтрации

⚠️ Расхождение: seed-схема не содержит полей title и category, которые использует страница. Схему нужно обновить через Admin UI.
