Дашборд/Компоненты/Card

01 · Card · Базовый

Универсальный контейнер с фоном, скруглением и бордером/тенью. Основа для ProgramCard, TeacherCard, ReviewCard, SeatsCard.

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/Card.astro → Showcase: /styleguide

Варианты

Вариант Когда Фон Бордер Тень Дефолт
surface По умолчанию --color-surface 1px solid --color-primary-line --shadow-sm, hover → --shadow-md ✓
elevated Акцент, важный элемент --color-surface none --shadow-lg
muted Приглушённый, неактивный --color-surface-alt none none
dark На тёмном фоне прозрачный 1px solid rgba(255,255,255,0.2) none

Использование (Astro)

---
import Card from '../components/ui/Card.astro';
---

<Card>Содержимое</Card>
<Card variant="elevated">Важное</Card>
<Card variant="muted">Мест нет</Card>
<Card variant="dark">На тёмном фоне</Card>

CSS-справка

Это справочник. Единая точка правды — src/components/ui/Card.astro.

.card              /* корень */
.card--elevated    /* вариант: усиленная тень */
.card--muted      /* вариант: приглушённый */
.card--dark       /* вариант: на тёмном фоне */

Ключевые токены:

  • padding: --spacing-2xl (3rem) → мобилка --spacing-xl (2rem)
  • border-radius: --radius-xl (1rem)
  • transition: --transition-base

Состав

Card — базовый, не собирается из других. Является основой для:

  • ProgramCard = Card surface + Badge ×2 + description + Link
  • TeacherCard = Card surface + Avatar + name + position
  • ReviewCard = Card surface + quote + author footer
  • SeatsCard = Card surface (или muted) + class name + spots count
