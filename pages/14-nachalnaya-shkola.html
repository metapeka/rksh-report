Уникальные стили

Класс Назначение
.ns-hero Hero-секция, text-align center, padding 4xl/2xl
.ns-hero__facts Flex-ряд для 3 fact-блоков (gap 2xl)
.ns-fact__val Playfair Display, 2xl, bold, color primary
.ns-fact__label sm, muted
.ns-about Секция особенностей, bg surface-alt
.ns-about__grid Grid 2×2 для карточек (gap lg)
.ns-about__card Карточка: bg surface, border primary-line, radius lg, padding xl
.ns-about__card-title Inter, semibold, lg, color primary
.ns-about__card-desc sm, muted, line-height relaxed
.ns-faq-section Секция FAQ, padding 4xl
.ns-faq-group Группа FAQ-вопросов (margin-bottom 2xl)
.ns-faq__item border-top/bottom primary-line
.ns-faq__q::before "+" / "−" prefix, color accent-warm
.ns-docs__item::before "✓" prefix, color success
.ns-cta CTA-баннер, bg primary, color surface

Адаптив (≤768px)

  • .ns-hero → padding 2xl/lg
  • .ns-about, .ns-faq-section → padding 2xl
  • .ns-about__grid → grid-template-columns 1fr
  • .ns-cta → padding 2xl
  • .btn внутри .ns-cta → width 100%
