Уникальные стили

Класс Назначение

.klass-hero__facts Flex-ряд для 3 fact-блоков (gap 2xl)
.klass-fact__val Playfair Display, 2xl, bold, color primary
.klass-fact__label sm, muted
.klass-program Секция программы, bg surface-alt
.klass-tag Pill-тег предмета (border-radius 9999, border primary-line, bg surface)
.klass-program__extra Centered текст «Дополнительно»
.klass-schedule__scroll overflow-x: auto для таблиц
.klass-table Общая таблица (schedule + prices): border-collapse, primary-line borders
.klass-table th bg primary, color surface, semibold
.klass-table tr:nth-child(even) td bg surface-alt
.klass-table__price nowrap, bold, color accent-warm
.klass-prices Секция тарифов, bg surface-alt
.klass-prices__link a color accent, semibold
.klass-faq__item border-top/bottom primary-line
.klass-faq__q::before "+" / "−" prefix, color accent-warm
.klass-cta CTA-баннер, bg primary, color surface

Адаптив (≤768px)

  • .klass-hero → padding 2xl/lg
  • .klass-program, .klass-schedule, .klass-faq, .klass-prices → padding 2xl
  • .klass-cta → padding 2xl
  • .btn внутри .klass-cta → width 100%
