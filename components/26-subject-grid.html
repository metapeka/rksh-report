26 · SubjectGrid — предметная сетка

Отображает список учебных предметов с часами. 3 варианта визуализации × 3 стиля иконок.

Расположение

src/components/ui/SubjectGrid.astro

Пропсы

Проп Тип Обязательный По умолчанию Описание
items { name: string; hours: number }[] ✅ — Список предметов
variant `'cards' 'bars' 'bubbles'` ❌
iconStyle `'outline' 'filled' 'duotone'` ❌
duotoneBg `'primary' 'neutral' 'none'` ❌
maxHours number ❌ auto (max из items) Максимум для шкалы
class string ❌ — Дополнительный CSS-класс

Варианты

A · Cards (variant="cards")

Сетка карточек с иконкой, названием и часами. grid-template-columns: repeat(auto-fill, minmax(140px, 1fr)).

  • A1: iconStyle="outline" — прозрачный фон иконки, stroke
  • A2: iconStyle="filled" — фон --color-primary-soft, solid fill
  • A3: iconStyle="duotone" — настраиваемый фон через duotoneBg

B · Bars (variant="bars")

Вертикальный список с прогресс-барами. Ширина пропорциональна hours/maxHours. Градиент: --color-primary → --color-accent-warm. max-width: --container-narrow.

C · Bubbles (variant="bubbles")

Круги разного размера (60–120px), flex-wrap с justify-content: center.
