15 · PackageTable · Страничный

Сравнительная таблица пакетных предложений: заголовок, подзаголовок, таблица с тарифами.

→ Код: src/components/ui/PackageTable.astro → Showcase: /styleguide

Состав

PackageTable = div.wrapper + h4.title + p.subtitle + div.scroll > table

Использование (Astro)

---
import PackageTable from '../components/ui/PackageTable.astro';
---

<PackageTable
  title="Пакетное предложение «Базовое»"
  subtitle="Занятия 2 раза в неделю. 8 занятий — 4 недели."
  tiers={[
    { label: '1 месяц', sessions: '8', perSession: '1 087 р.', total: '8 700 р.', savings: '0 р.' },
    { label: '4 месяца', sessions: '32', perSession: '979 р.', total: '31 320 р.', savings: '3 480 р. (10%)' },
    { label: '9 месяцев', sessions: '72', perSession: '924 р.', total: '66 555 р.', savings: '11 745 р. (15%)' },
  ]}
/>

Пропсы

Проп Тип По умолчанию Описание
title string — Заголовок таблицы (h4)
subtitle string — Подзаголовок (пояснение)
tiers Tier[] — Массив тарифных планов (обязательный)
class string '' Дополнительные CSS-классы

Tier

Поле Тип Описание
label string Название периода («1 месяц», «9 месяцев»)
sessions string Кол-во занятий
perSession string Цена за занятие
total string Итоговая стоимость
savings string Выгода. Если не «0 р.» — подсвечивается зелёным

Ключевые токены:

  • th: background: var(--color-primary), color: var(--color-surface)
  • th first-child: text-align: left
  • td first-child: text-align: left, font-weight: var(--font-weight-medium), background: var(--color-surface-alt)
  • total row: background: var(--color-surface-alt)
  • savings: color: var(--color-success) (зелёный)
