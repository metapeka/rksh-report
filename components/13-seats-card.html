Карточка свободных мест: класс + статус (есть места / мест нет).

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/SeatsCard.astro → Showcase: /styleguide

Состав

SeatsCard = Card (surface для available, muted для full) + label + spots

Варианты Card

status Card variant Внешний вид
available surface Белый фон, бордер, тень
full muted Приглушённый фон, без бордера, opacity 0.7

Использование (Astro)

---
import SeatsCard from '../components/ui/SeatsCard.astro';
---

<SeatsCard label="3 класс" status="available" spots="4 места" />
<SeatsCard label="2 класс" status="full" />

Пропсы

Проп Тип По умолчанию Описание
label string — Название класса (обязательный)
status 'available' | 'full' — Статус: есть места или нет (обязательный)
spots string 'Есть места' / 'Мест нет' Текст статуса. Автоподбор по status
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/SeatsCard.astro.

.scard              /* корень — Card, flex-column, centered */
.scard__label       /* название класса — heading, primary */
.scard__spots       /* статус — semibold, sm */
.scard__spots--yes  /* «есть места» — success */
.scard__spots--no   /* «мест нет» — error */
