Секции по порядку

  1. Hero — заголовок h1, описание, CTA-кнопка «Записаться»
  2. Все курсы — 2-колоночная сетка карточек курсов с бейджами (аналогично /doshkolniki)
  3. Стоимость курсов — центрированный блок со ссылкой на страницу цен

Паттерн

Страница следует тому же паттерну, что и /doshkolniki:

  • Классы с префиксом sh- вместо dk-
  • Та же сетка карточек, бейджи, кнопки
  • Тот же блок-линк на цены

Уникальные стили

Стили — scoped в shkolniki.astro (тег <style>). Классы с префиксом sh- по аналогии с dk-.

Класс Назначение
.sh-hero Hero-секция: padding 4xl/2xl, text-align center
.sh-hero__inner Ограничение ширины: --container-narrow
.sh-hero__lead Подзаголовок: font-size lg, muted text

.sh-courses__grid 2-col CSS grid, gap lg

.sh-course__link Ссылка «Подробнее →»: font-weight semibold, color accent

Медиа-запрос (≤768px)

  • Hero padding → 2xl/lg
  • Courses padding → 2xl
  • Grid → 1 колонка
  • Prices padding → 2xl
