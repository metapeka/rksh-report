Дашборд/Компоненты/Review card

12 · ReviewCard · Составной

Карточка отзыва: цитата + рейтинг + дата + автор + фото (или инициалы).

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/ReviewCard.astro → Showcase: /styleguide

Состав

ReviewCard = Card (surface) + quote + rating stars + date + footer (avatar/initials + author)

Использование (Astro)

---
import ReviewCard from '../components/ui/ReviewCard.astro';
---

<!-- Полный -->
<ReviewCard
  quote="«Мы с сыном ходили с 1.5 лет. Педагог оправдала все ожидания.»"
  author="Elena Mamonova"
  rating={5}
  date="2024"
  photoUrl="/uploads/mamonova.webp"
  photoAlt="Elena Mamonova"
/>

<!-- Без фото — покажет инициалы -->
<ReviewCard
  quote="«Отличная школа!»"
  author="Анна Смирнова"
  rating={4}
  date="2023"
/>

<!-- Минимальный -->
<ReviewCard
  quote="«Рекомендую!»"
  author="Игорь Петров"
/>

Пропсы

Проп Тип По умолчанию Описание
quote string — Текст отзыва (обязательный)
author string — Имя автора (обязательный)
photoUrl string — URL фото автора. Если пусто — инициалы
photoAlt string '' Alt-текст для фото
rating number — Оценка 1–5 (звёзды)
date string — Дата/период отзыва
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/ReviewCard.astro.

.rcard              /* корень — Card с font-style: normal */
.rcard__quote       /* blockquote — italic, text, line-height: relaxed */
.rcard__meta        /* flex row between stars and date */
.rcard__stars       /* rating stars — accent-warm, sm, letter-spacing */
.rcard__date        /* date — xs, muted */
.rcard__footer      /* footer — flex, avatar + name */
.rcard__avatar      /* img 48×48, circle, object-fit: cover */
.rcard__initials    /* span 48×48, circle, bg-accent-warm */
.rcard__author      /* name — primary, semibold, sm */

Ключевые токены:

  • avatar: 48×48px, border-radius: 50%
  • initials fallback: --font-size-lg, --font-weight-bold, white on --color-accent-warm
  • quote: italic, --line-height-relaxed
  • author: --font-size-sm, --font-weight-semibold, --color-primary
  • stars: --color-accent-warm, letter-spacing: 0.1em
  • date: --font-size-xs, --color-text-muted
