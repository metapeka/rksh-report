Дашборд/Компоненты/Gallery card

20 · GalleryCard · Страничный

Карточка фотографии для фотогалереи: изображение + подпись. Hover — зум + подъём.

→ Код: src/components/ui/GalleryCard.astro → Showcase: /styleguide

Состав

GalleryCard = div.gallery-card > div.image-wrap + p.title

Использование (Astro)

---
import GalleryCard from '../components/ui/GalleryCard.astro';
---

<GalleryCard
  title="Первый день школы"
  imageUrl="/uploads/photo1.jpg"
  imageAlt="Первый день школы"
/>

Пропсы

Проп Тип По умолчанию Описание
title string — Подпись под фото
imageUrl string — URL изображения
imageAlt string '' Alt-текст изображения
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/GalleryCard.astro.

 .gallery-card              /* корень: border, radius-lg, cursor-pointer */
 .gallery-card:hover        /* shadow-lg, translateY(-2px) */
 .gallery-card__image-wrap  /* aspect-ratio 4/3, overflow hidden, bg-alt */
 .gallery-card__image       /* 100%, object-fit cover, hover scale(1.05) */
 .gallery-card__placeholder /* flex center, muted icon */
 .gallery-card__title       /* sm, text, nowrap, ellipsis */

Ключевые токены:

  • border: 1px solid var(--color-primary-line)
  • border-radius: var(--radius-lg)
  • aspect-ratio: 4 / 3
  • hover image zoom: scale(1.05), transition-slow
  • title: --font-size-sm, nowrap, text-overflow ellipsis
