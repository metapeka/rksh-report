04 · Button · Базовый

Кнопка / ссылка-кнопка. 4 варианта через проп variant.

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/Button.astro → Showcase: /styleguide

Варианты

Вариант Когда Фон Текст Hover Дефолт
warm CTA, призыв к действию --color-accent-warm (#C88A3D) #fff --color-accent-warm-hover (#B07832) ✓
primary Основная, на светлых фонах --color-primary (#173B6E) #fff --color-primary-hover (#12315D)
outline Второстепенная transparent --color-primary bg → --color-primary-soft
social Соцсети, мелкие действия --color-primary-soft --color-primary bg → --color-primary, text → #fff

Использование (Astro)

---
import Button from '../components/ui/Button.astro';
---

<Button variant="warm" href="/kontakty">Получить презентацию</Button>
<Button variant="primary" href="/kontakty">Записаться</Button>
<Button variant="outline" href="/o-shkole">Подробнее</Button>
<Button variant="social" href="https://vk.com">VK</Button>

<!-- Без href — рендерится <button> -->
<Button variant="warm">Отправить</Button>

Пропсы

Проп Тип По умолчанию Описание
variant 'warm' | 'primary' | 'outline' | 'social' 'warm' Визуальный вариант
href string — URL. Если передан — рендерится <a>, иначе <button>
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/Button.astro.

.btn                 /* корень */
.btn--warm           /* CTA тёплая */
.btn--primary        /* CTA основная */
.btn--outline        /* Outline */
.btn--social         /* Соцсети */

Ключевые токены:

  • padding: var(--spacing-md) var(--spacing-2xl) (1rem 3rem), social: var(--spacing-sm) var(--spacing-lg)
  • border-radius: var(--radius-md) (0.5rem)
  • font-weight: semibold (600), social: medium (500)
  • transition: --transition-fast
  • Адаптив: warm/primary → width: 100% на ≤768px
