17 · DocumentList · Страничный

Список документов с названием, описанием, датой и кнопкой скачивания.

→ Код: src/components/ui/DocumentList.astro → Showcase: /styleguide

Состав

DocumentList = ul.doc-list > li.doc-list__item × N

Каждый элемент: заголовок + описание + дата + ссылка «Скачать» с иконкой.

Использование (Astro)

---
import DocumentList from '../components/ui/DocumentList.astro';

const docs = [
  { title: 'Устав', description: 'Устав образовательной организации', date: new Date('2024-01-15'), file: { src: '/uploads/ustav.pdf' } },
  { title: 'Лицензия', date: new Date('2024-03-01'), file: { src: '/uploads/license.pdf' } },
  { title: 'Приказ о зачислении', description: 'Приказ №12 от 01.09.2024' },
];
---

<DocumentList documents={docs} />

Пропсы

Проп Тип По умолчанию Описание
documents Document[] — Массив документов (обязательный)
class string '' Дополнительные CSS-классы

Document

Поле Тип Описание
title string Название документа (обязательный)
description string Краткое описание
date Date | null Дата документа
file { src?: string; alt?: string } | null Файл для скачивания
category string Категория
sveden_section string Привязка к разделу сведений

Если file.src отсутствует — ссылка «Скачать» не отображается. Если date null — дата не выводится.

CSS-справка

Это справочник. Единая точка правды — src/components/ui/DocumentList.astro.

.doc-list              /* ul: flex column, gap sm */
.doc-list__item        /* li: flex, justify-between, border, radius-md, padding md lg */
.doc-list__item:hover  /* border-color: accent, shadow-sm */
.doc-list__info        /* flex column, gap 2xs, min-width: 0 */
.doc-list__title       /* font-weight: medium, color: text, size: base */
.doc-list__desc        /* size: sm, color: muted, line-height: relaxed */
.doc-list__meta        /* flex row, align-items: center, gap md, flex-shrink: 0 */
.doc-list__date        /* size: sm, color: muted, nowrap */
.doc-list__link        /* inline-flex, bg-primary-soft, color: primary, radius: md */
.doc-list__link:hover  /* bg: primary, color: surface */

Ключевые токены:

  • border: 1px solid var(--color-primary-line)
  • border-radius: var(--radius-md)
  • hover: border-color: var(--color-accent), box-shadow: var(--shadow-sm)
  • download link: background: var(--color-primary-soft), hover → background: var(--color-primary)
  • Адаптив (≤768px): flex-direction: column
