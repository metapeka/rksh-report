03 · Badge · Базовый

Бейдж-метка. Компактная цветная плашка для обозначения возраста, вместимости, статуса.

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/Badge.astro → Showcase: /styleguide

Варианты

Вариант Когда Цвет текста Фон Дефолт
default Нейтральная метка (возраст, вместимость) --color-text-muted --color-primary-soft ✓
success Позитивный статус (есть места) --color-success #E8F5E9
error Негативный статус (мест нет) --color-error #FFEBEE

Использование (Astro)

---
import Badge from '../components/ui/Badge.astro';
---

<Badge>3–7 лет</Badge>
<Badge>до 6 человек</Badge>
<Badge variant="success">5 мест</Badge>
<Badge variant="error">Мест нет</Badge>

Пропсы

Проп Тип По умолчанию Описание
variant 'default' | 'success' | 'error' 'default' Визуальный вариант
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/Badge.astro.

.badge              /* корень */
.badge--success     /* зелёный */
.badge--error       /* красный */

Ключевые токены:

  • font-size: --font-size-sm (0.875rem)
  • padding: --spacing-2xs --spacing-sm (0.25rem 0.75rem)
  • border-radius: --radius-md (0.5rem)
  • success bg: #E8F5E9, error bg: #FFEBEE
