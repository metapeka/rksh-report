Источник данных

Данные Откуда
все данные Захардкожены в шаблоне (src/pages/srednyaya-shkola.astro)
Hero-факты Массив facts (возраст, группа, цена)
Причины Массив reasons (6 причин: title + desc)
Карточки особенностей То же, что в ns-about (4 карточки, общие стили)
Расписание (режим дня) Массив schedule (6 временных слотов)
Предметы Массив subjects (12 предметов: name + hours)
Каникулы Массив holidays (4 периода)

Данные не вынесены в CMS — обновляются вручную. Стили ns-about* — общие с /nachalnaya-shkola.

Адаптив (≤768px)

  • .ss-hero → padding 2xl/lg
  • .ss-reasons → padding 2xl
  • .ss-reasons__grid → grid-template-columns 1fr
  • .ss-about, .ss-schedule, .ss-subjects, .ss-holidays → padding 2xl
  • .ss-timeline__time → min-width 90px, font-size xs
  • .ns-about__grid → grid-template-columns 1fr
  • .ss-cta → padding 2xl
  • .btn внутри .ss-cta → width 100%
