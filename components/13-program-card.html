10 · ProgramCard · Составной

Карточка программы: заголовок, бейджи (возраст, размер группы), описание, ссылка.

→ Дизайн-гайд: ../page-specs/01-design-guide.md → Код: src/components/ui/ProgramCard.astro → Showcase: /styleguide

Состав

ProgramCard = Card (surface) + Badge × N + Link

Использование (Astro)

---
import ProgramCard from '../components/ui/ProgramCard.astro';
---

<ProgramCard
  title="Начальная школа"
  badges={["1–4 классы", "до 15 человек"]}
  description="Классическое образование для младших школьников по системе РКШ"
  linkUrl="/nachalnaya-shkola"
  linkText="Подробнее →"
/>

Пропсы

Проп Тип По умолчанию Описание
title string — Заголовок (обязательный)
badges string[] [] Массив текстов бейджей
description string — Описание программы
linkUrl string — URL ссылки
linkText string "Подробнее →" Текст ссылки
variant 'surface' | 'muted' | 'elevated' | 'dark' 'surface' Вариант Card
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/ProgramCard.astro.

.pcard              /* корень — обёртка Card */
.pcard__title       /* h3 — Playfair Display */
.pcard__badges      /* flex-контейнер для Badge */
.pcard__desc        /* описание — text-muted */
.pcard__link        /* ссылка — accent */

Ключевые токены:

  • title: --font-family-heading, --font-size-xl
  • badges gap: --spacing-sm
  • link: --color-accent, hover → --color-primary
