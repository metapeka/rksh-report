25 · PostCard

Карточка новости для списка и блока «Другие новости».

Расположение

src/components/ui/PostCard.astro

Пропсы

Проп Тип Обязательный Описание
title string ✅ Заголовок новости
href string ✅ Ссылка на детальную страницу
excerpt string ❌ Анонс (обрезается до 3 строк)
date string ❌ ISO-строка даты, форматируется в «10 апреля 2024 г.»
category string ❌ Название категории (отображается Badge)
image { id, src?, alt?, width?, height? } ❌ Объект изображения EmDash (через <Image>)
class string ❌ Дополнительный CSS-класс

Варианты

  1. С изображением — показывает 16:9 обложку поверх карточки
  2. Без изображения — карточка без блока изображения
  3. Без категории — не показывает Badge

Использование

<PostCard
  title={post.data.title}
  href={`/news/${post.id}`}
  excerpt={post.data.excerpt}
  date={post.data.publishedAt?.toISOString()}
  category={categoryLabel}
  image={post.data.featured_image}
/>

Стили

  • Фон: --color-surface, граница --color-primary-line
  • Hover: shadow-lg + translateY(-2px)
  • Изображение: object-fit: cover, zoom при hover
  • Excerpt: -webkit-line-clamp: 3
  • Адаптивность: на мобильных карточка растягивается на всю ширину
