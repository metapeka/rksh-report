HolidayCard

Карточка каникул — сетка 2× с эмодзи, названием и датами.

→ Общие правила: 00-guide.md

Props

Prop Тип Обязательное Описание
holidays Holiday[] ✅ Массив каникул
class string ❌ Дополнительный CSS-класс

Holiday

Поле Тип Обязательное Описание
name string ✅ Название (Осенние, Новогодние…)
dates string ✅ Даты (28 октября – 5 ноября)
icon string ❌ Эмодзи (🍂 🎄 ❄️ 🌱)

Варианты

Один вариант — сетка 2× с hover-эффектом.

По умолчанию

<HolidayCard holidays={[
  { name: "Осенние", dates: "28 октября – 5 ноября", icon: "🍂" },
  { name: "Новогодние", dates: "28 декабря – 8 января", icon: "🎄" },
  { name: "Зимние", dates: "15 февраля – 23 февраля", icon: "❄️" },
  { name: "Весенние", dates: "5 апреля – 13 апреля", icon: "🌱" },
]} />

Стили

Класс Назначение

.holiday-card__icon Эмодзи 2rem
.holiday-card__name Название, primary, semibold
