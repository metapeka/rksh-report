Дизайн-гайдлайны РКШ

Единые правила дизайна и вёрстки для всех страниц. Каждое ТЗ в page-specs/ ссылается сюда.

1. Типографика

Шрифты

Назначение Семейство CSS-переменная
Основной текст Inter (sans-serif) --font-family-base
Заголовки h1–h3 Playfair Display (serif) --font-family-heading

Размеры и веса

Элемент Размер Вес Цвет Примечание
h1 секции --font-size-4xl (2.25rem) 700 --color-primary Playfair Display
h2 секции --font-size-3xl (1.875rem) 600 --color-primary Playfair Display
h3 подзаголовок --font-size-xl (1.25rem) 600 --color-text Inter
Лейбл (Адрес, Телефоны...) --font-size-xs (0.75rem) 600 --color-text-muted uppercase, letter-spacing 0.06em
Основной текст --font-size-base (1rem) 400 --color-text line-height: 1.75
Текст описания --font-size-lg (1.125rem) 400 --color-text-muted line-height: 1.75
Кнопки --font-size-base (1rem) 600 зависит от типа —

Ссылки

  • Цвет: --color-accent (#2F6FB3)
  • Hover: --color-primary, underline
  • Переход: var(--transition-fast)

2. Отступы

Секции

Контекст Значение Пояснение
Padding секции (десктоп) --spacing-4xl (6rem) Вертикальный воздух
Padding секции (мобилка) --spacing-2xl (3rem) Экономия места
Margin h1 → контент --spacing-lg (1.5rem) Под заголовком
Margin intro → grid --spacing-2xl (3rem) После описания

Карточки

Контекст Значение Пояснение
Padding карточки (десктоп) --spacing-2xl (3rem) Внутренний воздух
Padding карточки (мобилка) --spacing-xl (2rem) Экономия
Gap между блоками внутри карточки --spacing-xl (2rem) Между строками контактов
Gap иконка → текст --spacing-lg (1.5rem) Рядом с SVG

Разделители и CTA

Контекст Значение Пояснение
Divider margin --spacing-sm (0.75rem) Сверху/снизу линии
CTA padding-top --spacing-lg (1.5rem) Отступ перед кнопкой

Grid

Контекст Значение Пояснение
Gap между колонками (десктоп) --spacing-3xl (4rem) Воздух между колонками
Gap между колонками (мобилка) --spacing-xl (2rem) Экономия

3. Иконки (SVG)

Правила

  • Формат: inline SVG
  • Цвет: currentColor → родительский элемент задаёт --color-primary
  • Stroke: stroke-width="1.5", stroke-linecap="round", stroke-linejoin="round"
  • Fill: none (outline-стиль), currentColor для лого соцсетей

Размеры

Контекст Width/Height margin-top
Контактные иконки (адрес, телефон) 22px 3px
Иконки в кнопках соцсетей 16px 0
Иконки в навигации 20px 0

Выравнивание

.icon {
  flex-shrink: 0;
  color: var(--color-primary);
  margin-top: 3px; /* выравнивание с первой строкой текста */
}

4. Карточки

.card {
  background: var(--color-surface);     /* #FFFFFF */
  border-radius: var(--radius-xl);      /* 1rem */
  box-shadow: var(--shadow-lg);         /* 0 10px 15px rgba(0,0,0,0.1) */
  padding: var(--spacing-2xl);          /* 3rem */
}

5. Кнопки

CTA (основная, тёплая)

.cta-button {
  display: inline-block;
  padding: var(--spacing-md) var(--spacing-2xl);   /* 1rem 3rem */
  background: var(--color-accent-warm);             /* #C88A3D */
  color: #fff;
  border-radius: var(--radius-md);                  /* 0.5rem */
  font-weight: var(--font-weight-semibold);         /* 600 */
  font-size: var(--font-size-base);                 /* 1rem */
  text-align: center;
  text-decoration: none;
}
.cta-button:hover {
  background: var(--color-accent-warm-hover);     /* #B07832 */
  transform: translateY(-1px);
}

Кнопки соцсетей

.social-link {
  display: inline-flex;
  align-items: center;
  gap: var(--spacing-sm);                           /* 0.75rem */
  padding: var(--spacing-sm) var(--spacing-lg);     /* 0.75rem 1.5rem */
  background: var(--color-primary-soft);            /* #DEE8F5 */
  color: var(--color-primary);                      /* #173B6E */
  border-radius: var(--radius-md);
  font-weight: var(--font-weight-medium);           /* 500 */
  font-size: var(--font-size-base);                 /* 1rem */
  text-decoration: none;
}
.social-link:hover {
  background: var(--color-primary);
  color: #fff;
}
