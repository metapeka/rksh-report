Секции по порядку

  1. Hero — заголовок h1, описание, CTA-кнопка «Записаться»
  2. Как проходит подготовка — 3 шага в карточках (выбор предмета → занятия → аттестация)
  3. Предметы и стоимость — таблица с 12 предметами, единообразной ценой и количеством занятий
  4. CTA-блок — призыв к записи с двумя кнопками

Уникальные стили

Стили — scoped в podgotovka-attestacii.astro (тег <style>). Префикс att-.

Класс Назначение
.att-hero Hero-секция: padding 4xl/2xl, text-align center
.att-hero__inner Ограничение ширины: --container-narrow
.att-hero__lead Подзаголовок: font-size lg, muted text
.att-info Секция шагов: bg --color-surface-alt, padding 4xl
.att-info__grid 3-col CSS grid для карточек шагов

.att-table__lessons Количество занятий: muted
.att-cta CTA-секция: padding 4xl, text-align center

Медиа-запрос (≤768px)

  • Hero padding → 2xl/lg
  • Info grid → 1 колонка
  • Table: горизонтальная прокрутка или карточки вместо строк
  • CTA padding → 2xl
