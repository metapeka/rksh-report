Расписание (/raspisanie)

Статус: ✅ Готова

→ Общие правила: 01-design-guide.md

Контент

Источник данных

Данные Откуда
Расписание 1–4 классов Захардкожено в шаблоне (src/pages/raspisanie.astro, массив schedule)
Ссылки на 0 класс / среднюю школу Захардкожены (href /0klass, /srednyaya-shkola)

Расписание не вынесено в CMS — обновляется вручную в шаблоне. Причина: простая структура (класс → время → список предметов).

Структура данных (хардкод)

schedule: Array<{
  grade: string;    // "1 класс"
  time: string;     // "9:00–12:20"
  subjects: string[]; // ["Физкультура", "Обучение грамоте", ...]
}>

4 записи: 1 класс (9:00–12:20, 7 предметов), 2 класс (9:00–13:10, 10 предметов), 3 класс (9:00–13:10, 9 предметов), 4 класс (9:00–13:10, 9 предметов).
