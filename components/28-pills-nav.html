28 · PillsNav — горизонтальная pills-навигация

Горизонтальный ряд кнопок-пилюль для переключения контента или фильтрации.

Расположение

src/components/ui/PillsNav.astro

Пропсы

Проп Тип Обязательный Описание
items { label: string; value: string }[] ✅ Список пунктов
active string ❌ Значение активного пункта
class string ❌ Дополнительный CSS-класс

Визуал

  • Неактивная пилюля: background: --color-surface, border: 1px solid --color-primary-line, color: --color-text-muted, border-radius: --radius-lg
  • Активная пилюля: background: --color-primary, color: --color-surface, border-color: --color-primary
  • Hover: border-color: --color-accent, color: --color-primary

Использование

<PillsNav
  items={[
    { label: 'Дошкольники', value: 'preschool' },
    { label: 'Школьники', value: 'school' },
  ]}
  active="preschool"
/>

Обычно используется вместе с JS-обработчиком кликов на родительской странице.
