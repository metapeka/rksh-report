30 · AccordionNav — аккордеон навигация

Раскрывающийся список пунктов с контентом внутри.

Расположение

src/components/ui/AccordionNav.astro

Пропсы

Проп Тип Обязательный Описание
items { title: string; content: string }[] ✅ Список пунктов (content — HTML-строка)
class string ❌ Дополнительный CSS-класс

Визуал

  • Заголовок: кликабельный, с шевроном, font-weight-medium, font-size-base
  • Шеврон: поворачивается на 180° при раскрытии
  • Контент: скрыт по умолчанию, раскрывается с transition: max-height
  • Граница снизу: 1px solid --color-primary-line

Особенность

Astro не поддерживает динамические имена слотов, поэтому контент передаётся пропом content (HTML-строка), а не через <slot>.

Использование

<AccordionNav
  items={[
    { title: 'Питание', content: '<p>Горячие обеды ежедневно...</p>' },
    { title: 'Медицинское обслуживание', content: '<p>Медсестра на месте...</p>' },
  ]}
/>
