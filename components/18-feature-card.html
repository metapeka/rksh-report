Состав

FeatureCard = div.card + icon-wrap + h4.title + p.desc

Использование (Astro)

---
import FeatureCard from '../components/ui/FeatureCard.astro';
---

<FeatureCard
  icon="M12 2L2 7l10 5 10-5-10-5z"
  title="Маленькие классы"
  description="Где каждому ребёнку уделяется время и внимание"

Пропсы

Проп Тип По умолчанию Описание
icon string — SVG path d для иконки 24×24. Если пусто — иконка не выводится
title string — Заголовок (обязательный)
description string — Описание
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/FeatureCard.astro.

.feature-card              /* bg-surface, border, radius-lg, padding-xl */
.feature-card:hover        /* shadow-md, border-color: accent */
.feature-card__icon-wrap   /* 3rem×3rem, bg-primary-soft, radius-md, flex center */
.feature-card__icon        /* svg 24×24, color: primary */
.feature-card__body        /* flex column, gap xs */
.feature-card__title       /* Inter, lg, semibold, color: text */
.feature-card__desc        /* sm, muted, relaxed */

Ключевые токены:

  • border: 1px solid var(--color-primary-line)
  • hover: box-shadow: var(--shadow-md), border-color: var(--color-accent)
  • icon-wrap: 3rem, background: var(--color-primary-soft)
  • Адаптив (≤768px): padding var(--spacing-lg)
