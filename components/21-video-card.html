Дашборд/Компоненты/Video card

21 · VideoCard · Страничный

Карточка видео для видеоархива: обложка + кнопка play + подпись. Клик — переход по внешней ссылке. Hover — зум + play-оверлей.

→ Код: src/components/ui/VideoCard.astro → Showcase: /styleguide

Состав

VideoCard = a.video-card > div.thumb (img + div.play) + p.title

Использование (Astro)

---
import VideoCard from '../components/ui/VideoCard.astro';
---

<VideoCard
  title="Экскурсия по школе"
  url="https://youtu.be/xxxxx"
  thumbnailUrl="/uploads/video1.jpg"
  thumbnailAlt="Экскурсия по школе"
/>

Пропсы

Проп Тип По умолчанию Описание
title string — Подпись под видео
url string — Ссылка на видео (открывается в новой вкладке)
thumbnailUrl string — URL обложки видео
thumbnailAlt string '' Alt-текст обложки
class string '' Дополнительные CSS-классы

CSS-справка

Это справочник. Единая точка правды — src/components/ui/VideoCard.astro.

 .video-card                /* корень: a, border, radius-lg, no underline */
 .video-card:hover          /* shadow-lg, translateY(-2px) */
 .video-card__thumb         /* aspect-ratio 16/9, relative, overflow hidden */
 .video-card__thumb-img     /* 100%, object-fit cover, hover scale(1.05) */
 .video-card__thumb-placeholder /* flex center, muted icon */
 .video-card__play          /* absolute inset, flex center, opacity 0→1 on hover */
 .video-card__title         /* sm, text, nowrap, ellipsis */

Ключевые токены:

  • border: 1px solid var(--color-primary-line)
  • border-radius: var(--radius-lg)
  • aspect-ratio: 16 / 9
  • play button: полупрозрачный круг с треугольником, появляется при hover
  • hover image zoom: scale(1.05), transition-slow
  • link: target="_blank", rel="noopener noreferrer"
